# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-22

### Added
- Initial release of Hugo Theme Daemon
- Apple-inspired minimalist design with clean white theme
- Responsive layout for mobile, tablet, and desktop
- Built-in search functionality with modal dialog
- Table of contents sidebar for articles
- Syntax highlighting with light color scheme
- Tag and category taxonomy pages
- Archives page organized by year
- Smooth scrolling and micro-interactions
- Backdrop blur effect on header
- Code block copy button
- Back to top button
- Social media links in footer
- Avatar and profile section on homepage
- Post navigation (previous/next)
- Reading time and word count display
- Feather icons throughout the theme
